# Telegram чат-бот для проведения розыгрыша

## 1. Стек
- Python 3.13.9
- Aiogram 3.22.0
- Asyncpg 0.30.0

Основа чат-бота реализована на Aiogram. Взаимодействие с СУБД PostgreSQL реализовано посредством асинхронной библиотеки Asyncpg.

## 2. Описание
Пользователь, имеющий код, который состоит из букв латинского алфавита и цифр, переходит в чат к боту и запускает его командой `/start`. Далее бот предлагает ввести код. После ввода программа проверяет код на валидность и активность — не был ли он использован кем-то ранее. Затем бот отправляет пользователю сообщение с результатом проверки.

После успешного использования кода в базе данных, в столбце `ActivatedByUserId` ("активировано пользователем с ID"), сохраняется идентификатор пользователя Telegram.
Количество кодов, которое может ввести пользователь, не ограничено.
